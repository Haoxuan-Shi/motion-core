# Security policy

## Supported versions

The latest release on the default branch receives security fixes.

## Reporting

Please report a suspected vulnerability privately through the repository
host's security-advisory feature. Include affected versions, a minimal
reproducer, impact, and any proposed mitigation. Do not include secrets or
private vehicle data.

This software is a simulation and prototyping component. It is not certified
for controlling vehicles or other safety-critical systems. Applications must
add independent input validation, watchdogs, fail-safe behavior, and hardware-
appropriate verification.
